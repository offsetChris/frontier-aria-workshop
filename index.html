WAI-ARIA Workshop

The activities on this site will present an accessibility code challenge that needs updating. You will modify the code however you see fit then activate submit to see the expected code result for the best practice of that ARIA concept. There is no strict validation and we're not keeping score so explore as you would like!

Interactive Activities by Topic:

  • Descriptions and Relationships
  • Landmarks
  • States and Properties
  • Composites
  • Live
