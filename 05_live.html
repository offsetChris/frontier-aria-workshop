Live Regions - WAI-ARIA Workshop

Activity 1

  1. Uses an aria-live property to create the live region.
  2. The live region announcements will notify the user immediately.
  3. The live region announces everything inside the region when an update to the content occurs.
Show the answer 
<div class="notifications" aria-live="assertive" aria-atomic="true">
  <p>This is a notification message for the user.</p>
</div>

Activity 2

  1. Notify users of a change during the next pause in user activity.
  2. Notify users only of additions to the list.
Show the answer 
<ul aria-live="polite" aria-relevant="additions">
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
  <li>Fourth item</li>
 </ul>

Activity 3

  1. Notifies users of changes immediately.
  2. Announces everything in the live region with every update.
Show the answer 
<div class="notifications" role="alert">
  <p>This is a notification message for the user using a role on my container.</p>
</div>
